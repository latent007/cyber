# BANDIT0 Writeup

## Starting the challenge
> Starting the challenge by opening a terminal and ssh to bandit.labs.overthewire.org on port 2220
---

### Solution
1. SSH with custom port.
- To SSH using other ports than the default port 22, use the following command.

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
- The password is the same as the username **bandit0**
